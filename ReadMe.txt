السلام عليكم يا حلوين

انتوا اخترتوا السهل معناته خلفيتكم هي:
- العناوين
- الفقرات
- التعليقات

افكار للموقع:
- موقع عن طريقه استخدام GitHub
- اي موقع في بالكم

متطلباتكم هيا التالي:
- استعمال رسوم العناوين بناء على الصوره<h1> <h2> .... <h6> => Images
- استعمال رسم الفقره <p> 
- اضف تعليقات بين الوسوم حتى تبين بداية و نهاية الوسوم => Images
- وجود 4 صفحات
- استخدام وسمين على الاقل للخطوط => Images

تذكير :
- لا تنسون استخدام html:5

تحذيرات :
- انتبه الى بدايه و نهاية الوسوم
- انتبه على ان يتشابه وسم البدايه و النهايه
- انتبه على الكتابه داخل وسم <body>
